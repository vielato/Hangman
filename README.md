# Hangman
Hangman Game Project in python.

##Using
[Python](https://www.python.org/)

## Guidance on running this project
1. Clone the project to your prefered folder.
2. Open terminal.
3. Navigate through project's folder.
4. Run `py game.py`
5. Let the game begin!
